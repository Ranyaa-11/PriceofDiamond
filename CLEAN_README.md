# 💎 Clean Diamond Price Predictor

**Ultra-simple diamond price prediction app - only what users need!**

## What's Included

- `clean_app.py` - Clean Streamlit app (40 lines)
- `clean_train.py` - Simple training script (30 lines)
- `simple_requirements.txt` - Minimal dependencies

## What's NOT Included (and why)

❌ **Training metrics** - Users don't need R² scores, MAE, RMSE  
❌ **Model comparisons** - Keep this in notebooks/reports  
❌ **Complex visualizations** - Users just want predictions  
❌ **Feature importance** - Internal evaluation only  
❌ **Data exploration** - Not needed for predictions  

## Quick Start

```bash
# Install
pip install -r simple_requirements.txt

# Train
python clean_train.py

# Run
streamlit run clean_app.py
```

## What Users Get

✅ **Clean input form** - Easy to use  
✅ **Instant predictions** - Real-time results  
✅ **Clear output** - Price and summary  
✅ **No clutter** - Only essential features
